What is Vim?

Vim is a highly configurable text editor built to enable efficient text editing. Vim is often called a "programmer's editor".

Vim's features

  • Persistent, multi-level undo tree.
  • Extensive plugin system.
  • Support for hundreds of languages and file formats.
  • Powerful search and replace.
  • Integrates with many tools.
